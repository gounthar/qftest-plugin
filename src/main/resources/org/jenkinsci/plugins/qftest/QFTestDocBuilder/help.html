Build Step

In this build step you can select the test-suites you want to generate the documentation for. (pkgdoc/testdoc)

Note that the path for a test-suite or parent folder has to be relative to the workspace, which has to be specified above under "Advanced Project Options - Use custom workspace".

Examples:
-"Suites/MySuite/some_suite.qft" only generates the documentation for one test-suite
-"Suites/MySuite" generates the documentation for all test-suites in the MySuite folder
-"*" generates the documentation for test-suites in your workspace

Any of these command line arguments under the label "Create test documentation" can be used for generating the documentation of your test-suites.
